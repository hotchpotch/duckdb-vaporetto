# v0.2.0

- Add DuckDB-Wasm release builds with embedded Vaporetto models.
- Add two Wasm model variants: a smaller `bccwj-suw_c1.0` build and a fuller
  `bccwj-suw+unidic_pos+kana` build with tag filtering support.
- Add the `keep_untagged` tokenizer option for keeping untagged tokens when
  `tags` filtering is enabled.
- Improve README introduction, Japanese overview, Wasm usage notes, and bundled
  model license notices.
